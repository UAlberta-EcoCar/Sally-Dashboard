## Contributing:

The firmware is an asynchronous embedded Rust project using the [Embassy](https://embassy.dev/) framework.
To get more familiar with Rust and Embassy, check out the following resources:

- Learning Rust: [The Rust Programming Language](https://doc.rust-lang.org/book/)
- Learning Embedded Rust: [The Embedded Rust Book](https://docs.rust-embedded.org/book/)
- Learning Embassy: [Embassy Book](https://embassy.dev/book/)
